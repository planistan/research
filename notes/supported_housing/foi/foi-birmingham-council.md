# FOI Request — Birmingham City Council (Supported/Exempt Accommodation)

To: Freedom of Information Team, Birmingham City Council  
From: a researcher  
Subject: FOI Act 2000 request — annual aggregate data on exempt/specified supported accommodation (2014 onward)

Dear FOI Team,

I am making this request under **section 1 of the Freedom of Information Act 2000**.

I am requesting **aggregate annual counts** (not individual-level data) for Birmingham, from **2014/15 onward** (or earliest available year), relating to exempt/specified supported accommodation.

Please provide the following in **CSV** format if possible (or tabular format if CSV is not available).

## Requested data (annual)

1. Total number of Housing Benefit claimants in exempt/specified accommodation.
2. The same claimant count, broken down by provider name (where recorded).
3. Claimant counts by commissioning type:
   - commissioned;
   - non-commissioned;
   - unknown/not recorded.
4. Claimant counts by main support-need category (where recorded).
5. Claimant counts with local connection to Birmingham vs no local connection (where recorded).
6. Claimant counts by previous local authority area or previous postcode prefix (for example outward code), where held in aggregate form.
7. Claimant counts by referral source category (where recorded), for example:
   - Birmingham council services;
   - other local authorities;
   - prison/probation;
   - hospitals/discharge routes;
   - asylum/refugee accommodation move-on;
   - self/provider referral;
   - other.
8. Counts of move-ins into exempt/specified accommodation from the following routes (where held):
   - asylum accommodation;
   - prison/probation;
   - hospitals;
   - other councils.
9. Average weekly **eligible rent** amount for exempt/specified claims.
10. Average weekly **eligible service charge** amount for exempt/specified claims.

## Format and scope notes

- Please provide this as annual totals by financial year where possible.
- I am requesting **aggregate data only**.
- If exact fields differ from wording above, please provide your closest equivalent and field definitions.

## Cost-limit and narrowing instruction (FOIA s.12)

If fulfilling all items would exceed the section 12 cost limit, please:

1. Process items in this order first: **(1), (2), (3), (5), (9), (10)**; and
2. Treat the remaining items as a narrowed follow-up request, or advise the exact reduction needed so the request can be refined.

I am content for this request to be handled as multiple linked requests if that is administratively easier.

Thank you.

Kind regards,  
a researcher
